## рџ› пёЏ How to Use This Repo

1. **Clone the repo**
   ```bash
   git clone https://github.com/singhdivyank09-coder/learn-python-with-projects.git
   cd learn-python-with-projects
   ```

2. **Set up a virtual environment** *(recommended)*
   ```bash
   python -m venv venv
   source venv/bin/activate      # Mac/Linux
   venv\Scripts\activate         # Windows
   ```

3. **Install dependencies** *(if any project requires them)*
   ```bash
   pip install -r requirements.txt
   ```

4. **Navigate to any project folder and run it**
   ```bash
   cd 02_control_flow/number_guessing_game
   python main.py
   ```
